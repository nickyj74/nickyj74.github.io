    Intended Use

    I have already indicated how these colours and styles will be used in the styling of any websites/pages so far, but here is a little round-up of just how they will be used.

    A little quick guide to the styles...

    • Background colour = Cornsilk - #F4F1F1 - RGB 255, 248, 220

    • h1 heading font colour = Maroon - #800000 - RGB 128,0,0

    • h2 heading font colour = Teal - #3CB371 - RGB 0,128,128

    • Main text colour = Black - #000000 - RGB 0, 0, 0
